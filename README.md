# 校园社团信息管理系统 java699

## 项目概述

校园社团信息管理系统是一款旨在方便校园内管理员、社长以及学生之间进行社团信息交流与管理的系统。本系统通过整合Springboot和Vue技术栈，实现了一整套前后端不分离的社团信息管理解决方案。

## 技术栈

- 后端：Springboot
- 前端：Vue2 + ElementUI
- 服务器环境：Nodejs14
- 数据库：Mysql5.7及以上
- JDK版本：1.8

## 功能模块

### 系统角色

- 管理员
- 社长
- 学生

#### 管理员角色功能

- **个人中心**：查看和修改个人信息。
- **学生管理**：管理学生信息，包括添加、修改、删除和查询。
- **社长管理**：管理社长信息，包括添加、修改、删除和查询。
- **社团分类管理**：社团分类的添加、修改、删除。
- **社团信息管理**：社团信息的管理，包括添加、修改、删除和查询。
- **加入社团管理**：审核学生加入社团的申请。
- **社团成员管理**：查看和管理社团成员信息。
- **社团活动管理**：审核社长创建的社团活动。
- **活动报名管理**：查看和管理社团活动的报名信息。
- **系统管理**：整体用户管理、权限管理、日志管理等。

#### 社长角色功能

- **添加社团**：创建新社团，填写信息并等待审核。
- **社团活动管理**：创建社团活动，填写活动信息并等待审核。
- **社团成员管理**：查看和管理本社团成员的信息。

#### 学生角色功能

- **加入社团**：提交加入社团申请，等待社长审核。
- **查看社团信息**：查看所有社团的详细信息。
- **参与社团活动**：报名参加社团活动，查看详情和报名状态。

## 运行环境

- JDK 1.8
- Mysql 5.7及以上
- 开发工具：IDEA、VSCode

## 目录结构

```plaintext
project/
├── backend/          # 后端代码目录

└── scripts/          # 数据库脚本目录
```

## 部署步骤

1. 安装JDK 1.8及Mysql数据库5.7及以上版本。
2. 解压项目代码至工作目录。
3. 导入数据库脚本，建立数据库环境。
4. 分别编译前后端代码，并启动后端服务。
5. 部署前端代码至Web服务器，并确保与后端服务通信正常。

- 前后端不分离，降低系统部署和维护复杂度。
- 基于Springboot和Vue的现代化技术栈，确保系统性能与开发效率。
- 系统角色明确，功能模块清晰，满足校园社团日常管理需求。
